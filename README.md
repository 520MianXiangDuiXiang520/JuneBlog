# JuneBlog 后端

## 1. 关于

这新型肺炎搞得，又出不去，又不想学习，就随便写个博客玩玩，还有好多 BUG 先上线了再说。 

[演示地址](https://junebao.top)

## 2. 功能

1. 在线文章增删查改
2. 评论
3. 每天通过邮件发送访客信息


## 2.感谢

* 前端首页和详情页模仿 [hexo 主题](https://github.com/litten/hexo-theme-yilia)
